Description

It allows you to save a new XML file or updating an existing XML file on your computer.

How to...

If you want to save an existing XML file you have to click on "Save" button from the buttons panel. After this a window will pop up where you have the possibility to navigate through your computer directories in order to save the file in a desired location.
