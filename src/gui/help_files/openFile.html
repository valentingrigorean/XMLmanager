Description

It allows you to open an existing XML file from your computer.

How to...

If you want to open an existing XML file you have to click on "Open" button from the buttons panel. If you have an opened file already, a message will pop up asking you if you want to save the current opened file. After that a window will pop up where you have the possibility to navigate through your computer directories in order to select the file you want to open.
