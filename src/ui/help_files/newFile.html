Description

It allows you to create a new XML file on your computer.

How to...

In order to create a new XML file you have to click on "New" button. If you have an opened file, a message will pop up asking you if you want to save the current opened file. After that a window will pop up where you have the possibility to navigate through your computer directories. After you decided where to create your new XML file, you have to enter a name for the file. You can do this by entering a name in the "File Name" field. After this you can now click on the "New" button where you will see that view panels are loading your XML file.
